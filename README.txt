CFvisualiser - CloudFormation Visualiser
----------------------------------------
Written By Chris Elleman (@chris_elleman)

Intended person is to be able to take a Amazon WebServices (AWS) CloudFormation
 JSON scripts and output a diagram in XML or even better SVG for import into
 presentations etc.


Things to watch for:

Webserver mime-type errors, the following types should be set:

visualiser.xslt - text/xml
